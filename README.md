# 📚 Student Study Planner

A simple and interactive Student Study Planner built using HTML, CSS, and JavaScript. This application helps students organize their study tasks, manage schedules, and stay productive.

## 🚀 Live Demo

https://ayushi-jain878.github.io/student-study-planner/

## ✨ Features

* Add study tasks
* Mark tasks as completed
* Delete tasks
* Responsive design
* Dark Mode support
* Simple and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📖 How to Use

1. Enter your study task.
2. Click the Add Task button.
3. Manage your tasks efficiently.
4. Mark completed tasks.
5. Delete tasks when no longer needed.

## 🎯 Project Objective

The purpose of this project is to help students plan their studies, track tasks, and improve productivity through better organization.
